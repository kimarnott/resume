Personal Projects

My DnD site

I'm quite proud of an environment I built for my DnD campaign using a low code platform. Considering I was trying to run an online homebrew campaign with word. Needless to say, that's wasn't the best plan.

My site has a bestiary (with links to more information on monsters/types etc), information on my very own made-up locations, characters (players and non-players), various organisations and groups in my world, maps, images, quests... etc. I also use it as a shared journal for myself and the players. There's a lot that's access controlled, a lot that my players will eventually 'unlock', and a whole lot of information.

My intent with the site is to store all my data in a separate database, probably PostgreSQL, as that's what I'm most familiar with (I'm currently using the inbuilt one until I get more cloud-savvy), so I can run some more cool/complex queries. And, eventually, code up a custom website myself (using what I'm learning from various web-dev courses).

Before you ask, I don't know copyright laws. And the info I've written myself is restricted to myself (otherwise the plot would be spoilt for my players), so the site is not public. But I'll happily show off my creation over a call.
